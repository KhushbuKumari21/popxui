# PopX React UI (Vite)

This project is a pixel-perfect React JS implementation of the provided Adobe XD design.

## Tech Stack
- React JS
- Vite
- React Router DOM
- CSS

## Features
- Mobile centered UI
- Landing, Login, Signup, Profile pages
- Clean routing
- Pixel-perfect layout

## Run Locally
npm install
npm run dev

## Build
npm run build
